# Animated 3D Website Experience

A stunning, production-ready animated website built with React, TypeScript, and cutting-edge web technologies. This project showcases advanced animations, 3D graphics, and seamless user interactions.

## 🌟 Features

### ✨ **Advanced Animations**

- **Framer Motion** powered page transitions and micro-interactions
- **3D CSS transforms** and GPU-accelerated animations
- **Scroll-based parallax** effects and viewport-triggered animations
- **Staggered animations** for dynamic content reveals
- **Interactive cursor follower** for enhanced user engagement

### 🎨 **Dual Theme System**

- **Nike-inspired mode**: Athletic, performance-focused design
- **Glasses mode**: Elegant, sophisticated eyewear showcase
- **Automatic theme switching** every 15 seconds
- **Smooth transitions** between design variants

### 🎯 **Interactive Components**

#### Navigation Header

- **Responsive design** with mobile-friendly hamburger menu
- **Animated search bar** with focus expansion effects
- **Hover animations** on all interactive elements
- **Smooth dropdown transitions**

#### Hero Section

- **3D glasses visualization** using Three.js
- **Animated text** with letter-by-letter reveals
- **Phase-based highlighting** system
- **Floating particle effects**
- **Dynamic typography** with custom font weights

#### Product Showcase

- **Nike mode**: Grid layout with hover effects and quick-add buttons
- **Glasses mode**: Carousel with rotation animations
- **Parallax scrolling** effects
- **Interactive product cards** with scale animations

#### Footer

- **Animated social icons** with hover effects
- **Newsletter signup** with form animations
- **Floating background particles**
- **Staggered content reveals**

### 🔧 **Technical Features**

#### Performance Optimizations

- **GPU acceleration** for smooth animations
- **Lazy loading** and viewport-based triggers
- **Optimized bundle size** with tree shaking
- **Smooth scrolling** with CSS optimizations

#### Responsive Design

- **Mobile-first approach** with adaptive layouts
- **Breakpoint-specific animations**
- **Touch-friendly interactions**
- **Cross-browser compatibility**

#### Accessibility

- **Semantic HTML** structure
- **ARIA labels** and proper roles
- **Keyboard navigation** support
- **Screen reader friendly**

## 🚀 **Technologies Used**

### Core Framework

- **React 18** - Modern React with hooks and concurrent features
- **TypeScript** - Type-safe development with full IDE support
- **Vite** - Lightning-fast development and build tool

### Animation & Graphics

- **Framer Motion** - Production-ready motion library
- **Three.js** - 3D graphics and WebGL rendering
- **@react-three/fiber** - React renderer for Three.js

### Styling

- **Tailwind CSS** - Utility-first CSS framework
- **CSS Variables** - Dynamic theming system
- **Custom animations** - Hand-crafted CSS keyframes

### UI Components

- **Radix UI** - Accessible component primitives
- **Lucide React** - Beautiful, customizable icons
- **Class Variance Authority** - Component variant management

## 🎮 **Interactive Elements**
